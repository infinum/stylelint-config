# stylelint-config

## Installation

NPM:
```bash
npm install @infinumjs/stylelint-config
```

Yarn:
```bash
yarn add @infinumjs/stylelint-config
```

## Generic support

```json
{
  "extends": "@infinumjs/stylelint-config"
}
```
